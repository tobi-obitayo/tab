# Tab Dashboard

A personal dashboard that replaces Chrome's new tab page.

## Development

Open `newtab.html` directly in any browser to develop and test changes without installing the extension.

## Load as Extension

1. Go to `chrome://extensions`
2. Enable **Developer mode** (toggle in the top-right)
3. Click **Load unpacked**
4. Select this folder (`tab/`)
5. Open a new tab — the dashboard will appear
